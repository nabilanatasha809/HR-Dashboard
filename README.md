#  Power BI HR-Dashboard

## Overview
This project presents a Human Resource Analytics Dashboard designed to monitor employee distribution, turnover trends, and workforce demographics across different factories and departments while helping teams make data-driven decisions.

## Tools
Microsoft Power BI,
Power Query,
DAX Measures,

## Features 
Employee Distribution,
Employee Demographics,
Employee Turnover Analysis,
